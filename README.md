# rust-c-ares

A Rust wrapper for the [`c-ares`](https://c-ares.haxx.se/) library, for
asynchronous DNS requests.

Most users should likely prefer
[`c-ares-resolver`](https://github.com/dimbleby/c-ares-resolver/), which offers
a much simpler API.
